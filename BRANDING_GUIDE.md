# Better Choice Insurance - Sales Tracking Platform

## 🎨 Brand Identity

### Visual Design
The Better Choice Insurance platform features a **professional, modern, and trustworthy** aesthetic designed specifically for insurance sales tracking.

**Design Philosophy:**
- **Trust & Reliability**: Clean layouts with subtle gradients and professional color palette
- **Performance-Focused**: Dashboard-first design emphasizing metrics and KPIs
- **Modern Professional**: Contemporary fonts (Sora + Outfit) paired with smooth animations
- **Accessible & Clear**: High contrast, readable typography, intuitive navigation

### Color Palette

**Primary (Brand Blue)**
- Main Brand: #0ea5e9 to #0369a1 (Gradient from light to dark blue)
- Creates trust, professionalism, stability
- Used for: Primary buttons, navigation, highlights

**Accent (Warm Gold)**
- #d97706
- Creates warmth and achievement feeling
- Used for: Awards, tier badges, special highlights

**Semantic Colors**
- Success: #10b981 (Active policies, completed)
- Warning: #f59e0b (Pending items)
- Danger: #ef4444 (Issues, cancelled)

### Typography

**Display Font: Sora** - Modern, geometric, professional
**Body Font: Outfit** - Clean, readable, friendly

## 🎯 Key Features

### 1. Branded Login Experience
Split-screen design with animated gradient background, feature highlights, and demo credentials

### 2. Dashboard
Personalized welcome, 4 key stat cards, recent sales feed, quick actions, tier display

### 3. Sales Management
Create sale modal, card-based list, PDF upload, status badges

### 4. Commission Tracking
Tier visualization, monthly breakdown, transaction history, tier structure

### 5. Statement Import (Admin)
Upload interface, import history, processing workflow

2. Run `./setup.sh`

Update branding in:

- `globals.css` - Fonts and animations
- Component files - Copy and messaging
